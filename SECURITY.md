# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.1.x | :white_check_mark: |
| 1.0.0.x | :x: |

## Reporting a Vulnerability

Report security vulnerabilities to security@playreia.com. Eventually you'll be able to file a report at https://playreia.com/contact/security.
